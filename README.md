# sample-docker-assignment

Install Docker Desktop for Windows
      Run the below commands to check docker version
     docker --version
     docker compose version

Verify Docker is Working
      Run this command: docker run hello-world
     If you see a success message, Docker is working correctly.

Clone the Project 
git clone https://github.com/lavanya-nagarro/sample-docker-assignment.git

Navigate to the sample-docker-assignment directory in the Docker CLI.
Start the application by running the command below, which will build and launch the frontend, backend, and database services:
docker compose up --build
 

To open the Application user the url  http://localhost:3000/
 
Action: Enter Username/Email → Click: Create User → Verification: Click Find User.
